對於移動

1. cd [folder/path] 進入 //cd ~(主頁面)、/(根目錄)、..(上一層)、.(目前位置)
2. ls //列出當前資料夾的所有檔案
3. pwd //查看當前路徑

對於檔案

1. vi、vim [檔案名稱] 後 按"a" or "i" 進入編輯模式 //修改檔案內容；完成後按ESC;輸入":";輸入w(wirte 確定寫入);輸入q(quite 保存後離開);q!為強制退出
2. echo [Message] '>' 或 '>>' [file name] //'>'為覆蓋先前資料改成輸入的資料，'>>'為在最後加上Message //echo是讓電腦重複我輸入的內容 ex.Echo hello 就會將hello原封不動輸入
3. cat、more、head、tail [file name] //查看文字檔內容，分別為快速查看、慢速查看、看前十行、看尾十行
4. touch [file name] //創建新檔
5. gedit [file name] //用記事本開啟檔案
6. mkdir [file name] //(make directory)建立新資料夾
7. cp [original path][new path] //複製檔案至新地方
8. cp -r [original path][new path] //複製資料夾 (r是recursive)
9. rm [file name] //刪除檔案remove
10. rm -r [file name] //刪除資料夾
11. mv [original file name][new file name] //將檔案移至資料夾 或是 改檔名
